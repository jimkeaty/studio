# **App Name**: BrokerView

## Core Features:

- Agent Daily Tracker: Agents log daily metrics (calls, engagements, appointments, contracts). Editable for 7 days.
- Business Plan Engine: Agents set annual income goal, app calculates required metrics (closings, contracts, etc.) broken down daily/monthly.
- Agent Dashboard/Report Card: Displays letter grade based on pace vs plan, progress bar, KPI tiles, net earned/pending, projected closings, estimated net income.
- Broker Command Dashboard: Aggregated view of agent performance, funnel metrics, closed/pending net income totals.
- Role-Based Access Control: Firebase Authentication with role-based access (agent, manager, broker, admin).
- Multi-Tenancy: Data isolation for each brokerage using brokerageId.
- Performance Forecasting Tool: AI powered forecast model to incorporate monthly commitments into the agents goals as they try to reach for the yearly income goals, tool analyzes daily and monthly data in order to provide up to date plans for each individual agents progress.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and professionalism.
- Background color: Light Gray (#F0F2F5) for a clean, modern look.
- Accent color: Orange (#FF9800) for call-to-action buttons and important highlights.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and readable interface.
- Use simple, clear icons from a library like Material Icons to represent different metrics and actions.
- Use a responsive grid layout to ensure the dashboard looks good on desktop and mobile devices.
- Subtle animations on data updates to draw the user's attention to changes.
